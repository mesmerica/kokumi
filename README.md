# kokumi
---
Customize flavortown your way! No code shareable flavortown theming. Hide anything you like, change fonts, colors, and more!
---
# features
- reveal hidden achievements
- achievements tab
- hiding a lot of stuff (anything in your sidebar, small things like the chef hat, suggestion player etc 
- theming (import any font you want from google fonts, change colors, and customize it your way)
- import/export of features and themes
---
# how to install
go to the [releases tab](https://github.com/mesmerica/kokumi/releases/latest) and download the latest one.
two ways: you can either load it unpacked OR you can load it in as a .crx file. both are easy, but chrome might flag the crx file 

### option A: unpacked
download the source code, go to chrome://extensions (or wherever your browser manages extensions), enable dev mode (topright), and select the kokumi folder

### option B: crx file
download the crx file from the releases tab, go to chrome://extensions (or wherever your browser manages extensions), drag and drop and you're done
